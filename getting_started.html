Getting started

Are you here for cookiecutting?

  1. Do you already have a Python package you want to standardize with scikit-package? Please follow the full instructions here.

  2. Do you already have a scikit-packaged package and you want to use the latest version? Please follow the instructions in here.

  3. Do you want to start a new Python repository with scikit-package? Start from the Cookiecutter workflow section here.

Are you here for release?

You already have a package standardized with scikit-package. Do you want to release your package to GitHub, PyPI, and conda-forge by creating a tag to your GitHub repository? Start from here.
